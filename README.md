# personal-website
Personal website designed using Figma and created with React/CSS.

This project was my first time using React, JavaScript, and CSS.

Beginning with the design process, I came up with and illustrated assets that I felt accurately displayed my personality and aesthetic. Then, I integrated the built-in assets from Figma into React where I used CSS to position them according to my creative vision.

Some concepts applied are using npm packages, flexbox, grid, and media querry sizing optimization.
